Take a picture
Classify it

[ WARNINGS ]
This Web App is a prototype of an image classifier. It classifies an image into 1000 categories (list in the ImageNet Kaggle competition) displaying the 3 most probable ones with their respective probabilities.
It uses the resnet50 layers version of the Deep Learning model that won the ImageNet competition in 2015 (ILSVRC2015).
As a prototype, this Web App could run slowly and as the ImageNet model has been trained with 1000 categories, some pictures could not be well labeled (for example, the mango category does not belong to these 1000 classes).
